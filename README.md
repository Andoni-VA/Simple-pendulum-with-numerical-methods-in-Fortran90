# Simple-pendulum-in-Fortran90

The objective of this repository is to resolve the equation for the simple pendulum and to make simulation using different methods. On one hand, we are using the Euler method, which isn't optimal, because the error is accumulative. On the other hand, we have the much more reliable method of Runge-Kutta 4.

# Modules

1. rk4 :: derivation method of Runge-Kutta 4
2. mcf_tipos :: we use it to set the precision
